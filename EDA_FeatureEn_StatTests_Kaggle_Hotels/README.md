### 2. Задачи проекта    

* на основании шаблона ноутбука посторить ML модель 
* проанализировать и преобразовать данные для качественной работы модели
* засабмитить решение в соревнование на Kaggle

**Что практикуем**     
* Очистка данных.
* Исследование данных.
* Генерация признаков.
* Отбор признаков.
* Преобразование признаков.
